Security Policy
Reporting a Vulnerability

The security of AdMaply and its users is important.

If you discover a security vulnerability, please do not open a public issue.

Instead, report it responsibly by sending an email to:

i3mc@i3mc.eu

Please include as much detail as possible:

A clear description of the issue

Steps to reproduce (if applicable)

Potential impact

Any suggested fixes (optional)

You will receive confirmation that your report was received. Valid reports will be investigated as quickly as possible.

Responsible Disclosure

We ask that you:

Do not publicly disclose the vulnerability until it has been addressed.

Avoid accessing, modifying, or deleting data that does not belong to you.

Avoid actions that could negatively impact users or system availability.

We appreciate responsible disclosure and your effort to help improve the security of this project.
